# 네이버 블로그 통계 분석기

네이버 블로그 방문자 통계를 한눈에 확인하고 분석할 수 있는 Chrome 확장 프로그램입니다.

## 주요 기능

- 오늘/어제/이번 주/이번 달 방문자 수 확인
- 전일 대비 방문자 변화율 표시
- 최근 7일 방문자 추이 차트
- 인기 게시글 TOP 5 확인
- 통계 데이터 JSON 파일로 내보내기
- 블로그 관리 페이지 바로가기

## 설치 방법

1. 이 폴더를 원하는 위치에 저장합니다.
2. Chrome 브라우저를 열고 주소창에 `chrome://extensions`를 입력합니다.
3. 우측 상단의 **개발자 모드**를 활성화합니다.
4. **압축해제된 확장 프로그램을 로드합니다** 버튼을 클릭합니다.
5. `chrome-extension` 폴더를 선택합니다.
6. 확장 프로그램이 설치되면 Chrome 툴바에 아이콘이 나타납니다.

## 사용 방법

1. 네이버 블로그 관리 페이지(https://admin.blog.naver.com)에 접속합니다.
2. Chrome 툴바에서 확장 프로그램 아이콘을 클릭합니다.
3. 팝업 창에서 방문자 통계와 인기 게시글을 확인할 수 있습니다.
4. **통계 데이터 내보내기** 버튼으로 데이터를 JSON 파일로 저장할 수 있습니다.

## 참고 사항

- 이 확장 프로그램은 **개인 사용 목적**으로만 제작되었습니다.
- 네이버 블로그에 로그인된 상태에서만 정상 작동합니다.
- 개인정보를 수집하거나 외부로 전송하지 않습니다.
