# 🔍 네이버 블로그 지수 분석기

실제 네이버 블로그 데이터를 크롤링하여 블로그 지수를 분석하는 도구입니다.

## 📋 분석 항목

| 항목 | 설명 | 배점 |
|------|------|------|
| 방문자 수 | 일일 방문자 수 | 30점 |
| 이웃 수 | 블로그 이웃 수 | 20점 |
| 포스팅 수 | 총 포스팅 개수 | 20점 |
| 스크랩 수 | 글 스크랩 횟수 | 15점 |
| 블로그 연차 | 블로그 운영 기간 | 15점 |

## 🏆 지수 등급

| 점수 | 등급 |
|------|------|
| 85+ | 최적1+ |
| 75-84 | 최적1 |
| 65-74 | 최적2 |
| 55-64 | 준최1 |
| 45-54 | 준최2 |
| 35-44 | 준최3 |
| 25-34 | 준최4 |
| 0-24 | 일반 |

---

## 🚀 실행 방법

### 1. 필요한 패키지 설치

```bash
pip install flask flask-cors requests beautifulsoup4 lxml
```

### 2. 서버 실행

```bash
python blog_analyzer_server.py
```

### 3. 브라우저에서 접속

```
http://localhost:5000
```

### 4. 블로그 분석

- 블로그 아이디 입력 (예: `mardukas`)
- 또는 전체 URL 입력 (예: `https://blog.naver.com/mardukas`)
- "분석하기" 버튼 클릭

---

## 📡 API 사용법

### 블로그 분석 API

```
GET /api/analyze?blog_id={블로그아이디}
```

**요청 예시:**
```bash
curl "http://localhost:5000/api/analyze?blog_id=mardukas"
```

**응답 예시:**
```json
{
  "blog_id": "mardukas",
  "blog_name": "먹자, 마시자, 놀러 다니자",
  "neighbors": 183773,
  "total_posts": 1500,
  "daily_visitors": 5000,
  "index": {
    "grade": "최적1+",
    "score": 92.5,
    "color": "#FFD700"
  }
}
```

## ⚠️ 주의사항

1. **방문자 수 조회**
   - 블로그 소유자가 방문자 위젯을 공개 설정해야 조회 가능합니다.
   - 비공개인 경우 0으로 표시됩니다.

2. **크롤링 제한**
   - 과도한 요청은 네이버에서 차단될 수 있습니다.
   - 적절한 간격을 두고 사용해주세요.

3. **지수 계산**
   - 이 지수는 자체 알고리즘으로 계산한 값입니다.
   - 네이버 공식 지수가 아닙니다.
